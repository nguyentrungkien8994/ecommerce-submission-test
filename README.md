How to run
1. cd ECOMMERCE.SUBMISSION.DATA
2. dotnet tool install --global dotnet-ef (Ignore if installed)
3. dotnet ef database update
4. Set ECOMMERCE.SUBMISSION.ASPRISE.APPHOST.AppHost as StartUp Project
5. Press F5 to run
